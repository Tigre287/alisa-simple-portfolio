# Alisa Aey - Portfolio

A modern, responsive portfolio website built for an Application Support Specialist. The site showcases professional experience, technical skills, and key automation/RPA projects.

## 🚀 Features

- **Modern Dark Theme**: Sleek design utilizing a dark color palette with cyan accents.
- **Responsive Design**: Fully responsive layout that adapts to mobile, tablet, and desktop screens.
- **Smooth Scrolling**: Integrated JavaScript for smooth navigation between page sections.
- **Project Showcase**: Dedicated section highlighting key automation and API integration projects.
- **Contact Information**: Easy access to email and GitHub profile.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure and content.
- **CSS3**: Custom styling, CSS variables for theming, and Flexbox/Grid for layout.
- **JavaScript (Vanilla)**: DOM manipulation and smooth scrolling functionality.
- **Google Fonts**: Uses the 'Inter' font family for clean, readable typography.

## 💻 Local Development

To view the website locally:

1. Clone or download this repository.
2. Open the `index.html` file in any modern web browser.
   - *Alternatively*, you can use a local server like Python's built-in HTTP server:
     ```bash
     python3 -m http.server 8000
     ```
     Then open `http://localhost:8000` in your browser.

## 🌐 Deployment

This project is designed to be easily hosted on **GitHub Pages**.

1. Push the code to a GitHub repository.
2. Go to the repository **Settings** > **Pages**.
3. Select the `main` branch as the source.
4. Save and wait for the site to build and deploy.
